Estructura frontend básica

Carpetas creadas:
- `css/`  -> hojas de estilo (archivo `main.css`)
- `js/`   -> scripts (archivo `app.js`)
- `src/`  -> código fuente adicional (componentes, templates)
- `assets/` -> recursos estáticos (imagenes, fuentes)

Archivo principal:
- `index.html` -> plantilla base que carga `css/main.css` y `js/app.js`

Notas:
- Los mockups HTML existentes están en `mockups/` y pueden usarse como páginas o componentes.
- Próximos pasos sugeridos:
  1. Convertir `mockups/*.html` en páginas dentro de `src/` o importarlas en `index.html`.
  2. Añadir workflow de build (Vite/Parcel) si se desea.
  3. Agregar variables globales CSS y un sistema modular JS.
